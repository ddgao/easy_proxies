# Issue Tracker: GitHub

Issues and PRDs for this repository live in GitHub Issues at `ddgao/easy_proxies`. Use the `gh` CLI for issue operations from this checkout.

## Conventions

- Create issues with `gh issue create`.
- Read issues and comments with `gh issue view <number> --comments`.
- Apply or remove labels with `gh issue edit`.
- Close issues with `gh issue close`.
- When a skill says to publish to the issue tracker, create a GitHub issue.

## Pull Requests As A Request Surface

External pull requests are not a triage request surface. Triage repository issues only.
